Some modded rounds cards I made for fun

1. Airstrike
    "Additionally fires an airstrike every time you shoot."
	  - -30% Damage
	  - -50% Attack Speed
	  - +0.5s Reload Time

2. Ammo Regen
    "Regenerates ammo each second"
      - +20% Ammo per second
	- +5 Ammo
	- +25% Attack Speed
	- -30% Damage

3. Big Damage
     "Damage goes big."
      - +50% Damage
      - +0.25s Reload Time

4. Blocked
     "You have been blocked."

      - +100% Health
	  - -50% Damage
	  - -100% Attack Speed
	  
5. Bounce House

	  - +30% Bullet Speed
	  - -10% Damage per bounce

	  - +0.5s Reload Time
	  
6. Cosine
     "Your bullets follow a sine wave instead of a straight line."

7. Shaped Glass

8. Suppressive Fire
